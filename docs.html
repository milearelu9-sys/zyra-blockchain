Getting Started with Zyra

1. Installation

# Install Zyra Core and Client Libraries
npm install @zyra/core @zyra/client

# Install CLI Tools
npm install -g @zyra/cli

2. Quick Start

import { ZyraClient } from '@zyra/client';

const zyra = new ZyraClient({
    network: 'mainnet',
    apiKey: 'YOUR_API_KEY'
});

// Connect to wallet
await zyra.connect();

3. Create Your First Smart Contract

// MyToken.sol
pragma solidity ^0.8.0;

import "@zyra/contracts/token/ZRC20/ZRC20.sol";

contract MyToken is ZRC20 {
    constructor() ZRC20("MyToken", "MTK") {
        _mint(msg.sender, 1000000 * 10**decimals());
    }

4. Deploy Using Zyra CLI

// Deploy to testnet
zyra deploy MyToken.sol --network testnet

// Deploy to mainnet
zyra deploy MyToken.sol --network mainnet

5. Interact with Your Contract

const contract = await zyra.loadContract('YOUR_CONTRACT_ADDRESS');

// Send tokens
await contract.transfer('RECIPIENT_ADDRESS', '1000000000000000000');

// Check balance
const balance = await contract.balanceOf('ADDRESS');
console.log(`Balance: ${zyra.utils.formatUnits(balance, 18)} MTK`);
